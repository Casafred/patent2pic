Patent2Pic 首次使用教程
========================

一、配置 AI 服务
----------------

使用 Patent2Pic 生成专利分解图前，需要先配置 AI 服务。

步骤如下：

1. 点击软件右上角的齿轮图标 [⚙]，打开"设置"对话框

2. 在"API 配置"标签页中：

   (1) 选择服务商（推荐使用"智谱 (GLM)"，国内访问速度快）
   
   (2) 输入 API Key
       - 智谱 AI: https://open.bigmodel.cn 注册后获取
       - DeepSeek: https://platform.deepseek.com 注册后获取
       - OpenAI 兼容: 支持任意兼容 OpenAI API 格式的服务商

   (3) 选择模型
       - 智谱推荐: glm-4-flash（免费额度大，速度快）
       - DeepSeek推荐: deepseek-chat（性价比高）
       - OpenAI推荐: gpt-4o

   (4) 点击"测试连接"验证配置是否正确

3. 确认成功后，关闭设置对话框


二、输入权利要求并生成分解图
----------------------------

1. 在左侧面板的文本框中粘贴专利独立权利要求文本

2. 系统会自动识别多条权利要求，显示在列表中

3. 点击要生成图形的权利要求（蓝色高亮表示选中）

4. 点击"生成分解图"按钮

5. 等待 AI 分析完成，分解图将自动显示在画布上


三、快速提示
------------

• API Key 填写后会自动保存，下次启动无需重新输入
• 可在"提示词设置"中自定义 AI 分析的提示词
• 生成的图形支持拖拽调整位置、双击编辑文字
• 点击"导出"可保存为 PNG、SVG 或 JSON 格式
• 项目文件支持保存为 .p2p 格式，下次继续编辑


常见问题
--------

Q: AI 抽取失败怎么办？
A: 请检查：
   - API Key 是否正确
   - 网络连接是否正常
   - 选择的模型是否可用
   - 可以在设置中点击"测试连接"排查问题

Q: 提示"已保护你的电脑"？
A: 点击"更多信息" → "仍要运行"即可，这是 Windows 的正常安全提示。


如需更多帮助，请参阅《用户使用说明书》。
