## 简介

Patent2Pic 是一款面向专利从业人员的专业桌面工具，能够将专利独立权利要求文本**自动转换**为清晰、规范的可视化结构图（功能分解图）。借助 AI 大语言模型，软件可智能识别权利要求中的技术特征、部件及其相互关系，并自动生成图形表达，大幅提升专利分析的效率与准确性。

## 核心功能

### 智能解析
- 自动识别和解析专利独立权利要求文本
- 支持多条权利要求同时输入与分段识别
- 自动识别权利要求编号和引用关系

### AI 智能抽取
- 集成多种 AI 服务商：**智谱 AI (GLM)**、**DeepSeek**、**OpenAI 兼容接口**
- 智能提取技术特征、部件节点和结构关系
- 支持自定义提示词，优化抽取效果
- 支持**并行处理模式**，同时处理多条权利要求，大幅提升批量效率

### 可视化编辑
- 基于 AntV X6 的专业图形编辑器
- 节点拖拽、缩放、自动布局（DAG 算法）
- **全局字号调节**（10px ~ 28px），节点边框自动适应
- **边点击高亮**，快速识别关系路径
- **组合框（限定框）**，最高层级节点自动归组，支持显示/隐藏
- 丰富的右键菜单和双击编辑
- 多步骤撤销/重做

### 权利要求翻译
- **一句一对照**呈现原文与翻译
- 翻译与原文**高亮同步**，选中节点时对应术语同时高亮
- 支持独立配置翻译模型与目标语言（中/英/日/韩/德/法）
- 自动翻译与 AI 抽取**并行执行**，不增加等待时间
- 逐句并行处理，单句失败不影响其他句子

### 多格式导出
- **PNG** — 3 倍高清分辨率，适合插入文档和演示文稿
- **SVG** — 矢量图，内容居中，适合高质量打印
- **Excel** — 原文与翻译对照表，适合审阅和进一步编辑
- **JSON** — 结构化数据，包含节点、边、分组信息
- **项目文件** (.p2p) — 完整编辑状态，可重新打开继续编辑

### 多标签页管理
- 每条权利要求对应独立画布标签页
- 切换标签时自动同步左侧权利要求阅读区
- 关闭最后一个标签页时恢复空白画布
- 自动保存标签页状态，切换回来可恢复编辑内容

### 自动保存
- 每隔 30 秒自动保存当前状态
- 关闭软件时自动保存
- 下次启动自动恢复上次的编辑状态

## 界面预览

Patent2Pic 采用经典的三栏式布局：

| 区域 | 功能 |
|------|------|
| **左侧面板** | 权利要求文本输入、分段识别、对照阅读（原文+翻译）、导出 Excel |

## 安装与使用

### 安装版

1. 下载 `Patent2Pic_x.x.x_x64-setup.exe` 安装程序
2. 双击运行安装程序，按提示完成安装
3. 从桌面快捷方式或开始菜单启动 Patent2Pic

### 便携版

1. 下载并解压压缩包到任意目录
2. 双击 `patent2pic.exe` 运行，无需安装
3. 删除整个文件夹即可卸载

> **注意**：便携版中 `patent2pic.exe` 与 `patent2pic_lib.dll` 必须在同一目录下。

### 首次运行

Windows 可能弹出"已保护你的电脑"安全提示，点击 **更多信息** → **仍要运行** 即可。

## 快速入门

```
配置 AI 服务 → 输入权利要求文本 → 点击"生成分解图" → 编辑调整 → 导出保存
```

1. **配置 AI** — 点击标题栏齿轮图标，在 API 配置中填入 API Key 并选择模型
2. **输入文本** — 在左侧面板粘贴专利独立权利要求文本
3. **生成图形** — 点击"生成分解图"按钮，AI 自动分析并生成可视化图形
4. **编辑调整** — 拖拽节点、调整样式、全局字号调节、自动布局
5. **导出保存** — 导出 PNG/SVG/Excel 或保存为 .p2p 项目文件

## 图例说明

### 节点类型

| 类型 | 颜色 | 含义 |
|------|------|------|
| 部件 | 蓝色 | 独立技术部件或组成部分 |
| 子系统 | 橙色 | 由多个部件组合而成的子系统或功能模块 |
| 特征 | 绿色 | 部件的具体技术特征、属性或参数 |

### 关系类型

| 类型 | 线型 | 箭头 | 含义 |
|------|------|------|------|
| 位置关系 | 实线 | 实心三角 | 空间位置关系（如"位于""设置在"） |
| 动作关系 | 实线 | 实心三角 | 功能动作关系（如"驱动""连接"） |
| 包含关系 | 虚线 | 空心三角 | 层级包含关系（如"包括""包含"） |
| 逻辑关系 | 点划线 | 菱形 | 条件/因果逻辑关系（如"当...时"） |

### 组合框

最高层级（层级 0）的节点若具有包含关系，其子节点会自动归入**组合框**（橙色虚线框）。组合框支持显示/隐藏、手动添加，拖动内部节点时框体自动调整大小。

## 开发构建

### 环境准备

- Node.js 18+
- Rust（通过 [rustup](https://rustup.rs/) 安装）
- pnpm 或 npm

### 开发运行

```bash
# 安装依赖
npm install

# 开发模式运行
npm run tauri dev
```

### 构建发布

```bash
# 构建生产版本
npm run tauri build
```

构建产物位于 `src-tauri/target/release/bundle/` 目录。
